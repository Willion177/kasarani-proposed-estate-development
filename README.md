Proposed Estate Map – Quick Guide

Welcome! This is an interactive map showing the proposed estate development in Kasarani. You can explore different areas, see details, and get a sense of the layout. No GIS degree required.

How to Use the Map

Pan & Zoom

Click and drag to move around the map.

Scroll or pinch to zoom in and out.

Layers & Features

Use the layer menu to show/hide roads, plots, or amenities.

Click on features (plots, roads, facilities) to see details like name, size, or type.

Search

Type a location or plot number in the search box to jump straight to it.

Legend

Colors and icons on the map represent different types of land, buildings, and facilities. Check the legend in the corner for quick reference.

Tips

Hover over elements for quick info.

Click on popups for more detailed information.

Explore different layers to see planned vs. existing features.

About This Map

Built with Mapbox GL JS.

Data includes roads, plots, green spaces, and amenities.

Fully interactive for easy exploration.

Support

If something doesn’t look right or isn’t working, just reload the page or check the layer toggle.
